# Changelog

Notable user-facing changes are documented here.

## 0.1.1 - 2026-07-16

### Added

- Conversation search by contact name or phone number.

## 0.1.0 - 2026-07-16

### Added

- Read-only browsing of recent macOS Messages conversations.
- Contact-name resolution from the local Contacts database.
- Message paging and Markdown exports for configurable time ranges.
